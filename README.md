# Wp-yw-Social-Medi-w-na-ycie-Program-Python
Projekt bada korelacje między czasem w Social Media, snem i stresem a wynikami studentów (oceny, skupienie). Aplikacja Python (Tkinter/Pandas) liczy statystyki (IQR, Pearson) i wizualizuje 6 kluczowych zależności za pomocą 5 typów wykresów w każdej zakładce.
